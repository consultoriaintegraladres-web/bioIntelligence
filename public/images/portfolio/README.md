# Imágenes del Portafolio

Esta carpeta contiene las imágenes que se mostrarán en los slides automáticos de la página de inicio.

## Estructura de archivos esperada:

- `bioretail-consultoria.jpg` - Imagen para el slide de BIORETAIL Consultoría
- `oportunidad-soat.jpg` - Imagen para el slide de Oportunidad SOAT
- `evolucion-auditoria.jpg` - Imagen para el slide de Evolución de Auditoría
- `adres-pagador.jpg` - Imagen para el slide de ADRES Pagador Inteligente
- `plataforma-soat.jpg` - Imagen para el slide de Plataforma SOAT
- `beneficios-taas.jpg` - Imagen para el slide de Beneficios TAAS

## Formato recomendado:

- **Formato**: JPG o PNG
- **Resolución**: Mínimo 1920x1080px (Full HD)
- **Tamaño**: Máximo 2MB por imagen
- **Aspecto**: 16:9 (landscape) para mejor visualización

## Cómo subir imágenes:

1. Coloca las imágenes en esta carpeta (`public/images/portfolio/`)
2. Asegúrate de que los nombres coincidan con los definidos en `PortfolioSlides.tsx`
3. Las imágenes se cargarán automáticamente en los slides

## Nota:

Si una imagen no está disponible, el slide se mostrará solo con el gradiente de fondo y el texto.
